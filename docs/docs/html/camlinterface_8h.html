AML_EXPORT CAMLErrorCode AMLData_GetKeys ( amlDataHandle_t  amlDataHandle,
char ***  keys,
size_t *  keysSize 
)

This function returns a list of key that AMLData has.

Parameters
amlDataHandle[in] handle of AMLData.
keys[out] character array that contains keys.
keysSize[out] the size of 'keys'.
Return values
#CAML_OKSuccessful.
#CAML_INVALID_PARAMInvalid parameter.
#CAML_NO_MEMORYFailed to alloc memory to character array.
Note
Character array will be allocated to 'keys', so it should be freed after use. (See the below example) ex) for (size_t i = 0; i < keysSize; i++) free(keys[i]); free(keys);
